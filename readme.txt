Project management tracking for TrustyStudio clients.

== Description ==

Trusty Studio - Project Tracking is a plugin specifically created for TrustyStudio clients. It allows them to access various information directly from the WordPress dashboard. The plugin adds a new "Trusty Studio" tab which includes 4 sub-tabs: information, remaining maintenance package time tracking, contact, and direct email sending to TrustyStudio.

Main features:

    Allows TrustyStudio clients to track their project directly from their dashboard.
    Quick access to information, remaining time tracking, contact, and email sending.

== Third-party Service ==

This plugin relies on a third-party service provided by Trusty Studio to fetch the remaining time for your project. Data from your site, such as the sanitized site title, is sent to the service to retrieve the relevant information. The third-party service only stores non-sensitive data in a JSON file and does not process any sensitive information. To learn more about Trusty Studio and their GDPR compliance, please visit their official website:

Service: http://tma.trusty-projet.fr/
Trusty Studio Official Website: https://www.trustystudio.fr/
GDPR Compliance: https://www.trustystudio.fr/politiques/

== Installation ==

    Download and unzip the plugin's ZIP file.
    Upload the "trusty-studio-project-tracking" folder to the "/wp-content/plugins/" directory on your WordPress site.
    Activate the plugin from the WordPress dashboard by going to "Plugins" > "All Plugins" and clicking on "Activate" under the plugin's name.
    A new "Trusty Studio" tab will appear in the WordPress dashboard.

== Usage ==

To use the plugin, follow these steps:

    Log in to your WordPress site's dashboard.
    Click on the "Trusty Studio" tab in the left-hand menu.
    Navigate between the 4 tabs to accurately track your project.
